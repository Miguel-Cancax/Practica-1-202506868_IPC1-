# Pac-Man Game en Java

Este es un proyecto de un **juego de Pac-Man** en consola, desarrollado en **Java**. El objetivo es mover a Pac-Man por un tablero, recolectando premios y evitando fantasmas. El juego se juega a través de la consola y se ejecuta con las teclas indicadas para mover a Pac-Man.

## Requisitos

- **Java JDK** instalado en tu computadora. Si no tienes Java
- Un entorno de desarrollo como **NetBeans**
